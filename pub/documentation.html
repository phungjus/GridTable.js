Getting Started:

To begin using GridTable.js you need to place the following scripts inside your HTML document as follows:

<head>
<script defer type="text/javascript" src='gridtable.js'></script>
<link href="stylesheet.css" rel="stylesheet" type="text/css">
 </head>

Then inside the body tag of your HTML Code you must have a <div> tag with a unique ID for each new GridTable you create as follows:

<body>
<div id='uniqueID'></div>
 </body>

After completing the two steps above you are ready to start using the GridTable.js library, it is now simply just instantiating the GridTable object in your Javascript file as follows:

const newGridTable = new GridTable()

Constructors

newGridTable.makeGridTable(numRows, numCols, selector, uniqueID, gridOrTable)

This function takes in 5 parameters: numRows, numCols, selector, uniqueID, and gridOrTable. It then creates an HTML table with 'numRows' rows and 'numCols' columns and id: 'GridTable' + uniqueID, then adds that table to the HTML element with id: 'selector'. Then depending on whether a grid or table was selected different functions can be called.

Example Code:

newGridTable.makeGridTable(5, 5, "uniqueID", 'newTable', 'table') //This creates a table with ID: newTable that has 5 rows and 5 columns newGridTable.makeGridTable(5, 5, "uniqueID", 'newGrid', 'grid') //This creates a grid with ID: newTable that has 5 rows and 5 columns

newGridTable.insertData(rowNum, colNum, data)

This function takes in three parameters called cellRow, cellCol, data. The function inserts the new data into the cell located at row: cellRow and column: cellCol. This function can only be used with Tables not Grids

Note: this function will throw an error if the GridTable hasn't been initialized (i.e. no columns or no rows created) or if the GridTable object is a Grid as this function is only used for Tables

Example Code:

newGridTable.insertData(2, 2, 'Testing') //This inserts 'Testing' into the cell at row 2 column 2

newGridTable.checkInitialization()

This function returns whether or not a GridTable object has been created

Example Code:

newGridTable.checkInitialization()

newGridTable.rowFinder(colNum, cellData)

This function returns the row the cellData is located in based on the column number input: colNum

Example Code:

newGridTable.rowFinder(2, 'Testing')

newGridTable.colFinder(colHeader)

This function returns the column number given the name of the column header provided by the input colHeader

Example Code:

newGridTable.colFinder('Header 1')

newGridTable.updateRow(rowNum, updatedList)

This function takes in two parameters rowNum and updatedList. It then updates all the data in the row with the values inside the Array updatedlist.

Note: this function will throw an error if the length of updatedList doesn't match up to the number of columns inside the GridTable.

Example Code:

newGridTable.updateRow(1, ['Name', 'Quantity'])

newGridTable.updateCol(colNum, updatedList)

This function takes in two parameters colNum and updatedList. It then updates all the data in the column with the values inside the Array updatedlist.

Note: this function will throw an error if the length of updatedList doesn't match up to the number of rows inside the GridTable.

Example Code:

newGridTable.updateCol(1, ['Item ID', '001', '002'])

newGridTable.updateHeader(listOfNewHeaders)

This function takes in a parameter listOfNewHeaders. It then updates the header of the GridTable with the data inside the Array listOfNewHeaders.

Note: this function will throw an error if the length of updatedList doesn't match up to the number of columns inside the GridTable.

Example Code:

newGridTable.updateHeader(['Name', 'Quantity'])

newGridTable.deleteRow(rowNum)

This function takes in a parameter rowNum. It then deletes the specified row from the GridTable. You also cannot delete the row of header cells.

Note: this function throws an error if the rowNum to be deleted is the header, or isn't a valid row number in the GridTable

Example Code:

newGridTable.deleteRow(2)

newGridTable.deleteCol(colNum)

This function takes in a parameter colNum. It then deletes the specified column from the GridTable.

Note: this function throws an error if the column to be deleted isn't a valid number (i.e. colNum is negative or colNum isn't in the GridTable object)

Example Code:

newGridTable.deleteCol(1)

newGridTable.sortData(colNum, direction)

This function takes in two parameters colNum and direction. It then sorts the specified column in accordance to the direction specified which can either be: 'asc' for ascending order or 'desc' for descending order.

Example Code:

newGridTable.sortData(2, 'asc')

newGridTable.addRow(rowData)

This function takes in a parameter rowData. It then adds a row to the GridTable where each entry is taken from the rowData Array.

Note: this function throws an error if the length of rowData is not equal to the number of columns in the GridTable object.

Example Code:

newGridTable.addRow(['NewValue1', 'NewValue2'])

newGridTable.addCol(colData)

This function takes in a parameter colData. It then adds a column to the GridTable where each entry is taken from the colData Array.

Example Code:

newGridTable.addCol(['Cell1', 'Cell2', 'Cell3'])

newGridTable.rowSwap(rowIndexOne, rowIndexTwo)

This function takes in two parameters rowIndexOne and rowIndexTwo. It then swaps those two rows in the table.

Note: This function is only to be used with Tables not Grids.

Example Code:

newGridTable.rowSwap(1, 2)

newGridTable.colSwap(colOneIndex, colTwoIndex)

This function takes in two parameters colOneIndex and colTwoIndex. It then swaps those two columns in the table.

Note: This function is only to be used with Tables not Grids.

Example Code:

newGridTable.colSwap(1, 2)

newGridTable.filter(queryVariable, queryOperator, queryComparison, insertID)

This functions takes in 4 parameters queryVariable, queryOperator, queryComparison, and insertID. It then filters the data according to the three query variables, and inserts a new table with the filtered data in the HTML element with id: insertID

Note: there are only 6 valid queryOperators: ==, !=, <, >, <=, >=

Note: queryVariable can only be the names of the header cell of each column.

Note: This function can only be used with Tables not Grids.

Example Code:

newGridTable.filter('Item Price', '<=', '500', 'filterResult')

newGridTable.addEvent(eventObj, cellRow, cellCol)

This function takes in three parameters eventObj, cellRow, and cellCol. This function can only be called with the Grid form of GridTable object, it adds the eventObj to the specified cellRow and cellCol.

Example Code:

newGridTable.addEvent({name: 'EventName', details: 'EventDetails', tag: 'EventTag'}, 2, 1)

newGridTable.deleteEvent(eventName, eventTag, cellRow, cellCol)

This function takes in two integers called cellRow and cellCol and two strings called eventName and eventTag, it then deletes the event located at that position based on eventName.

Note: the parameter eventTag is case-sensitive. It can be: 'None' if there isn't a event tag associated to the event.

Example Code:

newGridTable.delete('EventName', 2, 1)

newGridTable.modifyEvent(updatedEventObj, cellRow, cellCol)

This function takes in an object called updatedEventObj, it must contain the keys: name and details. It will then update the event at the specified row and column. This function can only be used with Tables

Example Code:

newGridTable.modifyEvent({name: 'UpdatedEventName', details: 'UpdatedEventDetails', tag: 'UpdatedEventTag'}, 2, 1)

newGridTable.groupTags(tagName, insertValue)

This function creates a new table that shows all the events that have the tagName associated with them.

Note: that the valid tagNames are case-sensitive along with the tag: 'all' which shows all events.

Note: this function can only be used with Grids not Tables

Example Code:

newGridTable.groupTags('all', 'displayResults')

newGridTable.exportGridTable()

This function takes no inputs, it will return a Object which represents the GridTableJS object.

Example Code:

newGridTable.exportGridTable()

newGridTable.importGridTable(gridTableObject)

This function takes in an object called GridTableObject, it is an object which has been exported by the GridTableJS library to specifically re-create the previously saved GridTable.

Example Code:

const exported = newGridTable.exportGridTable() newGridTable.importGridTable(exported)
