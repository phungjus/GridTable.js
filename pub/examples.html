Additional Features:

The user can easily sort each column by simply clicking on the header cell of the column, click once for ascending order and again for descending order

The user can also re-arrange columns by clicking on the columns header cell and dragging it left and right to the place you wish for that column to be

The user can also re-arrange rows by clicking on each rows first cell from the left and dragging it up and down to the place you wish for that row to be
